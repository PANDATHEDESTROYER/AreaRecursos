Definicion De Logaritmo

Definicion el logaritmo es una operacion relacionada con la potencia ya que consiste en hallay un valor denominado logaritmo dado la base del logaritmo y el argumento en breve se vera su definicion formal

Expresion : logba=c Si  bc = a

Ejemplo : Dada la siguiente expresion log39=2 compruebe que es correcta

Proceso : log39=2 ,   b=3,a=9,c=2  si c=2 entonces bc=a  32=9 si es

Propiedades De Los Logaritmos

1-Propiedad De La Base Igual Al Argumento

Expresion : logaa=1

Ejemplo : compruebe que log55=1

Proceso : Si log55=1 entonces 51=5 ↔5=51 ∴  log55=1

2-Propiedad Logaritmo De Uno

Expresion : logb1=0

Ejemplo : compruebe que log41=0

Proceso : Si log41=0 entonces 40=1 ↔ 40 = 4242=42-2=40=1616=1 ∴ log41=0

3-Propiedad Producto Suma

Expresion : logbx·y = logbx+logby

Ejemplo :compruebe que  log232·4 = log232+log24

Proceso : log232·4 = log232+log24↔ log2128 = 5+2↔7 = 7 Si

 Si log232·4=7 y  log232+log24 = 7 entonces log232·4=log232+log24

4-Propiedad Divicion Resta

Expresion : logbyx = logby-logbx

Ejemplo : compruebe que log2644 = log264-log24

Proceso : log2644 = log264-log24 ↔  log216 = 6-2↔4=4

Si log2644 =4 y  log264-log24 =4 entonces  log2644 = log264-log24 

5-Propiedad Potencia Factor Por Logaritmo

Expresion : logban = n · logba

Ejemplo : compruebe que log392 = 2 · log39

Proceso : log392 = 2 · log39↔ log381= 2 · 2  ↔  4 = 4

Si log392 = 4 y 2 · log39 =4  entonces log392 = log39

6-Propiedad Del Argumento Igual A La Base

Expresion : logbbn = n 

Ejemplo : compruebe que log553 = 3 

Proceso : log553 = 3 ↔ log5125=3 Si 53=125  entonces  log553 = 3

7-Propiedad De La Raiz

Expresion : logban = 1n logba

Ejemplo : compruebe que log3812 = 12 log381

Proceso : log3812 = 12 log381 ↔ log39 = 12 · 4↔ 2 = 42 ↔2=2

Si log3812 = 2 y 12 log381 = 2 entonces  log3812 = 12 · log381

8-Propiedad Cambio De Base

Expresion : logba = logcalogcb

Ejemplo : Dado  log28 cambie la base del logaritmo de 2 a 5

Proceso : log28=3 Por que 23 = 8 ↔ log58log52 = 1.2920...0.4306... = 3 = log523log52 = 3 · log52log52 = 3·11=31=3

9-Propiedad De La Cadena

Expresion : logda = logba·logcb·logdc

Ejemplo : compruebe que  log327 = log527·log85·log38

Proceso : log327 = log527·log85·log38 ↔ 3 = 2.0478...·0.7739...·1.8927... = 3
