En esta seccion tendra acceso a algunos de los proyectos liberados en github con el proposito de dar una idea de los elementos que maneja el autor de este sitio, actualmente son 3 proyectos de tres areas muy importantes

Aplicacion Movil : Programado en android studio este proyecto es de una app que simula muebles virtuales en la camara que da una idea de una remodelacion sin tener que mover o tener el mueble en fisico eso si requiere cargarse el dise√±o 3D en la app

Sitio Web : Programado en VisualCode este sitio web simula un sitio de ventas online con la idea de tener una tienda virtual con inventario virtual que de querer pasarse a un sitio de produccion ya funcional requerira de de una conexxiona una base de datos de algun almacen o almacenes ya mapeados

Programa De Escritorio: Echo en Visual Studio este programa cuenta con una interfaz grafica que controla servomoteres remotamente, nota para ver su funcionalidad debe contar con un arduino una camara y demas drivers para que pueda conectar el programa en remoto las especificaciones se dan en el repositorio github
